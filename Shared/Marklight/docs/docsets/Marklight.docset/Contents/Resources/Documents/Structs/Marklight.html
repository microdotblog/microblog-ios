Marklight

public struct Marklight

Marklight struct that parses a String inside a NSTextStorage subclass, looking for markdown syntax to be highlighted. Internally many regular expressions are used to detect the syntax. The highlights will be applied as attributes to the NSTextStorage’s NSAttributedString. You should create your our NSTextStorage subclass or use the readily available MarklightTextStorage class.
